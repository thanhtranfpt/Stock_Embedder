# Stock_Embedder
Stock Embedder of The Finance Forecasting Project


# Guide to Training

**Step 1**. Install: `pip install -r requirements.txt`

**Step 2**. Config in the file: `src/train_model/config.json`

**Step 3**. Check the file: `python src/train_model/main.py` (*if needed*)

**Step 4**. Run in terminal: `python src/train_model/main.py --config_file=src/train_model/config.json`
